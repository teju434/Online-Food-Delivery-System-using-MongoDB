MongoDB Food Ordering System

📌 Overview:

A MongoDB-based database project that simulates a food ordering system.
It demonstrates CRUD operations, filtering, and aggregation queries across multiple collections.

🗂️ Collections:

Restaurants – restaurant details

Menu – food items and pricing

Orders – customer orders and status

Payments – payment methods and transactions

DeliveryAgents – agent details and ratings

Reviews – customer feedback and ratings

⚙️ Key Features:

🔍 Querying with filters (find, conditions, sorting)

📊 Aggregation using $group, $avg, $sum, $max, $min

📈 Analytics like:

    Revenue calculation

    Top customers & restaurants

    Rating analysis

    City-wise insights

🛠️Tech Stack

    MongoDB

    JavaScript (Mongo Shell)

📈 Outcome

    Hands-on experience with MongoDB queries

    Understanding of aggregation pipelines

    Real-world data analysis use cases
